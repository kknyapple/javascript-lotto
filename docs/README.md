## **📄** 기능 목록

- 로또 발행하기
  - 숫자인지 확인
  - 1,000원 단위 인지 확인 (예외 처리)
  - 로또 개수 구하기
- 로또 번호 만들기
  - 1-45까지 숫자 인지 확인 (예외 처리)
  - 6개 랜덤한 숫자인지 확인 (예외 처리)
  - 중복 숫자 확인(예외 처리)
  - 오름차순 정렬
- 당첨 번호 입력 받기
  - , 기준으로 구분
  - 1-45까지 숫자 인지 확인 (예외 처리)
  - 6개의 랜덤한 숫자인지 확인 (예외 처리)
  - 중복 숫자 확인 (예외 처리)
- 보너스 번호 입력 받기
  - 1-45까지 숫자 인지 확인 (예외 처리)
  - 당첨 번호와 중복 숫자 확인 (예외 처리)
- 로또 번호와 당첨 번호 비교하기
- 일치 내역에 따른 수익률 계산하기
  - 소수점 둘째 자리에서 반올림

\*\*예외 처리 - 에러 문구는 throw문 [ERROR]로 시작해야 함
